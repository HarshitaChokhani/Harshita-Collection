# Harshita Collection Boutique

An elegant online boutique offering handcrafted Indian traditional wear for women. Harshita Collection brings together timeless ethnic fashion, curated collections, and a seamless online shopping experience.

## Live Website

https://harshita-collection.lovable.app

## About

Harshita Collection is a digital boutique focused on traditional Indian fashion, featuring sarees, suits, kurtis, dupattas, festive collections, and curated seasonal edits for modern women. The platform provides customers with a smooth browsing and shopping experience while enabling efficient store management through an admin dashboard. :contentReference[oaicite:1]{index=1}
